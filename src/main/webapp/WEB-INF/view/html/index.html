宠物托运检疫证明办理流程及注意事项

一、宠物具备动物卫生监督所或授权委托的宠物医院开具的狂犬疫苗《免疫证》，且宠物疫苗接种时间应在20天以上一年以内。
二、省级动物卫生监督机构指定动物检疫实验室出具的《宠物检测报告》。
2.1带上宠物和《免疫证》前往动物检疫实验室办理《宠物检测报告》。
2.2动物检疫实验室对宠物进行抽血化验，检查是否具有疫苗抗体。
2.3费用：290元/只 办理时间：约1小时 《宠物检测报告》有效期：7天
三、带上宠物、办理人身份证、《免疫证》、《宠物检测报告》、前往动物卫生监督所申报办理《动物检疫证明》。
费用：免费 办理时间：约0.5小时 《动物检疫证明》有效期：5天

因相关证件含有效期，请根据机票航班合理安排办证时间,由于各省市对宠物《动物检疫证明》的办理要求不同，有返程需求的客户请提前做好异地办理攻略。
